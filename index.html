Comando de Linux/centOS-VMware Workstation

Comandos útiles para trabajar con Linux y VMware Workstation.

Navegación y archivos

Comando pwd: Muestra la ruta de la carpeta actual.

Comando cd: Cambia el carpeta actual.

Comando ls -la: Lista todos los archivos y carpetas en la carpeta actual, incluyendo los ocultos.

Crear y manipular archivos

Comando mkdir: Crea una nueva carpeta.

Comando touch: Crea un nuevo archivo.

Comando cat: Muestra el contenido de un archivo.

Comando nano: Edita un archivo de texto. (abre un editor de texto)(Presiona Ctrl+X para guardar y salir)

Copiar, mover y eliminar archivos

Comando cp: /nombre de archivo/ /nombre del archivo copia/ : Copia archivos o carpetas.

Comando mv: /nombre del archivo copia/ /ruta donde copiara/: Mueve archivos o carpetas.

Comando rm: /ruta del archivo/: Elimina un archivo.

Búsqueda y filtrado

Comando grep: "palabra" /ruta del archivo/: Busca un palabra en un archivo.

Comando find: /ruta /nombre del archivo: Busca un archivo en una ubicación específica.(-name ".txt" 2>/dev/null muestra todo los txt sin importar permiso)

Permisos y procesos

Comando chmod: Cambia los permisos de un archivo o carpeta.

Comando ps aux: Muestra la información de los procesos en ejecución.

Comando top: Muestra los procesos en ejecución con información en tiempo real.(Presiona q para salir)

Información del sistema

Comando df h: Muestra el espacio en disco disponible.

Comando free -h: Muestra la cantidad de memoria libre y en uso.

Información del usuario y sistema

Comando whoami: Muestra el nombre del usuario actual.

Comando uname -a: Muestra información detallada sobre el sistema operativo.

Historial de comandos

Comando history: Muestra el historial de comandos.
